# O2Jam Interface Editor

WARNING: THIS IS STILL IN ALPHA! A lot of features are either not fully implemented or not implemented at all, and there are many bugs.

------------
An application that allows you to edit the interface files of O2Jam (opi, ojs, bnd, etc.)

### Features:
- OJS Editor
- BND Editor
- OPI State/Interface Viewer
- ControlList_Interface Support
- Compatible with most O2Jam clients
- abusedata.ojs editor
- and more!

Made in Godot 4.6.1, with some code transformed from the Python package O2Tools.

# Building

- Download Godot 4.6.1 here: https://godotengine.org/download/windows/
- Download the code for the interface editor as a .zip file, and extract.
- Import the project into Godot.
- Click Project -> Export and add the preset for your operating system.
- Download the build files when asked, and press export project to export as an EXE.
